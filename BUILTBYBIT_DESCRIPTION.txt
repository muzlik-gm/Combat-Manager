[SIZE=6][B][FA=fas fa-info-circle][/FA] What is True Combat Manager?[/B][/SIZE]

True Combat Manager is a comprehensive combat management plugin for competitive Minecraft servers. It tracks every aspect of player fights with detailed statistics, enforces intelligent restrictions to prevent unfair advantages, protects against combat logging, and presents everything through a beautiful, customizable interface.

Built for performance with async processing, intelligent lag compensation, and thread-safe architecture. Perfect for practice servers, factions, KitPvP, or any PvP-focused gameplay.

[SIZE=5][B][FA=fas fa-star][/FA] Why Choose This Plugin?[/B][/SIZE]
[LIST]
[*][B]Stop combat logging[/B] with instant-death protection and inventory drops
[*][B]Prevent unfair escapes[/B] with smart cooldowns (pearls, gapples, elytra, teleports)
[*][B]Track player performance[/B] with lifetime statistics (wins, losses, K/D, damage)
[*][B]Review fights[/B] with the combat replay system
[*][B]Customize everything[/B] - 6 themes, 6 sound profiles, 100+ config options
[*][B]Lag compensation[/B] automatically adjusts timers during server performance issues
[*][B]Safe zone protection[/B] prevents players from entering protected areas during combat
[*][B]Flexible cooldown system[/B] - enable/disable cooldowns outside of combat
[/LIST]

[SIZE=6][B][FA=fas fa-cog][/FA] Configuration[/B][/SIZE]

Highly configurable with 100+ options:

[CODE=yaml]
combat:
  duration: 30              # Combat timer (seconds)
  cooldown: 10              # Cooldown after combat
  max-sessions: 100         # Max concurrent sessions

restrictions:
  enderpearl:
    cooldown: 10
    cooldown-outside-combat: false  # NEW: No cooldown outside combat
    combat-cooldown-multiplier: 2.0
  golden-apple:
    cooldown: 3
    cooldown-outside-combat: false  # NEW: No cooldown outside combat
    combat-cooldown-multiplier: 1.5
  enchanted-golden-apple:
    cooldown: 8
    cooldown-outside-combat: false  # NEW: No cooldown outside combat
    combat-cooldown-multiplier: 2.0
  safezone:                         # NEW: Safe zone protection
    enabled: true
    block-entry: true
    protected-regions:
      - "spawn"
      - "safezone"
      - "safe"
    barrier:                        # NEW: Visual barrier system
      enabled: true
      material: "BARRIER"           # BARRIER, GLASS, RED_STAINED_GLASS
      height: 4
      width: 5
      duration-ticks: 6

visual:
  themes:
    default-theme: "clean"
    available: ["minimal", "fire", "ice", "neon", "dark", "clean"]
  sounds:
    profile: "default"

performance:
  lag:
    enabled: true
    tps-threshold: 18.0
  async:
    enabled: true
    thread-pool-size: 4

logging:
  storage:
    type: "BOTH"  # FILE, MEMORY, or BOTH
  summary:
    delivery: "CHAT"  # CHAT, GUI, STORAGE, or NONE
[/CODE]
